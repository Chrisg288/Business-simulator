# Economy Master Overview Prototype v1.3

Changes:
- Cleaned top sector row by removing Law/Rights, Infrastructure, Resources/Environment, and Information/Communication from the sector buttons.
  These are now treated conceptually as cross-sector overlays/layers rather than first-row sectors.
- Split Shelter → Rent and Shelter → Ownership into separate nodes.
- Selecting Human / Consumer → Shelter → Rent or rent subtypes now opens a practical rental listing/comparison data grid in the center panel.
- Selecting Shelter → Ownership opens a different ownership/affordability options grid.
- Rental grid intentionally echoes the original Comparator TreeView + DataGrid + lower transaction/detail-panel pattern.
- Life-domain wheel remains for domain-level nodes, but action/listing-level nodes now switch to the appropriate working representation.
